memory_resource

The type of memory resource used by the library.

Synopsis

Defined in header <boost/json/memory_resource.hpp> 

using memory_resource = boost::container::pmr::memory_resource;
Remarks

This alias is deprecated in favor of boost::container::pmr::memory_resource. It is included for backwards compatibility and shouldn't be used in new code. It will be removed completely in version 1.87.0.

Convenience header <boost/json.hpp>
