polymorphic_allocator

The type of polymorphic allocator used by the library.

Synopsis

Defined in header <boost/json/memory_resource.hpp> 

template<
    class T>
using polymorphic_allocator = boost::container::pmr::polymorphic_allocator< T >;
Remarks

This alias is deprecated in favor of boost::container::pmr::polymorphic_allocator. It is included for backwards compatibility and shouldn't be used in new code. It will be removed completely in version 1.87.0.
